# UPPM-PackStream
Universal Package Manager per Windows – repository centrale di pacchetti e installer

## Descrizione
UPPM-PackStream è la repository ufficiale per il package manager **UPPM**. Contiene tutti gli installer ufficiali disponibili per gli utenti tramite **UUPPM** e per gli sviluppatori tramite **PUPPM**. Questa repository serve come punto centrale per la gestione, distribuzione e aggiornamento dei pacchetti.

## Versioni di UPPM
- **UPPM Classic**: versione tutto-in-uno, include PUPPM e UUPPM  
- **PUPPM (Push Universal Package Manager)**: destinato agli sviluppatori per pubblicare applicazioni e pacchetti  
- **UUPPM (User Universal Package Manager)**: destinato agli utenti finali per scaricare e utilizzare pacchetti già pubblicati  
- **UGPM (Universal GNU Package Manager)**: compatibile con pacchetti GNU/Linux e ecosistemi open-source
